Política de Privacidad

Última actualización: 26 de octubre de 2025

Sistema de Gestión QR se compromete a proteger su privacidad. Esta política describe cómo recopilamos, usamos y protegemos su información personal cuando utiliza nuestra aplicación.

1. Información que Recopilamos

Nuestra aplicación recopila y procesa la siguiente información:

  • Datos Personales: Nombre completo, DNI, correo electrónico, número de teléfono
  • Datos Financieros: Montos de pagos y estado de empadronamiento
  • Imágenes: Fotografías de perfil (opcional, proporcionadas voluntariamente por el usuario)
  • Códigos QR: Códigos generados que contienen información de identificación
  • Datos de Autenticación: Tokens de acceso de Google (temporales y encriptados)
  • Estatus Mensual: Registro del estado mensual de cada cliente

2. Cómo Usamos Su Información

Utilizamos la información recopilada para los siguientes propósitos:

  1. Gestión de Usuarios: Administrar y mantener registros de clientes
  2. Generación de Códigos QR: Crear códigos QR únicos para verificación de identidad
  3. Sincronización de Datos: Sincronizar información entre Supabase y Google Sheets
  4. Verificación de Estado: Consultar y actualizar el estatus mensual de empadronamiento
  5. Mejora del Servicio: Analizar el uso de la aplicación para mejorar la experiencia del usuario

3. Almacenamiento y Seguridad de Datos

3.1 Supabase (Base de Datos Principal)

Todos los datos personales se almacenan de forma segura en Supabase, una plataforma de base de datos con las siguientes características de seguridad:

  • Encriptación de datos en tránsito (SSL/TLS)
  • Encriptación de datos en reposo
  • Autenticación y autorización basada en roles
  • Copias de seguridad automáticas
  • Cumplimiento con estándares de seguridad internacionales

3.2 Google Sheets (Sincronización Opcional)

Si el usuario autoriza la sincronización con Google Sheets:

  • Los datos se sincronizan únicamente cuando el usuario lo solicita explícitamente
  • La conexión utiliza OAuth 2.0 para autenticación segura
  • Solo se accede a las hojas de cálculo específicas autorizadas por el usuario
  • El usuario puede revocar el acceso en cualquier momento desde su cuenta de Google

3.3 Fotografías

Las imágenes de perfil se almacenan en el servicio de almacenamiento seguro de Supabase con:

  • Compresión automática para optimizar el tamaño
  • URLs públicas generadas de forma segura
  • Control de acceso mediante políticas de seguridad

4. Acceso a la API de Google

Uso Limitado de Datos de Google: El uso de información recibida de las APIs de Google se adherirá a la Política de Datos de Usuario de los Servicios de API de Google, incluidos los requisitos de uso limitado.

4.1 Alcances (Scopes) Solicitados

Nuestra aplicación solicita los siguientes permisos de Google:

  • https://www.googleapis.com/auth/spreadsheets - Para leer y escribir en Google Sheets

4.2 Cómo Usamos el Acceso a Google Sheets

El acceso a Google Sheets se utiliza exclusivamente para:

  • Sincronizar datos de clientes desde Supabase a Google Sheets
  • Facilitar la gestión y análisis de datos por parte del administrador
  • Crear copias de respaldo de información en formato de hoja de cálculo

4.3 Datos NO Compartidos con Terceros

Los datos obtenidos de Google APIs NO se utilizan para:

  • Publicidad o marketing dirigido
  • Perfilado de usuarios
  • Venta o transferencia a terceros
  • Entrenamiento de modelos de inteligencia artificial
  • Cualquier propósito no relacionado directamente con la funcionalidad de la aplicación

5. Compartir Información

NO compartimos, vendemos ni divulgamos su información personal a terceros, excepto en las siguientes circunstancias:

  • Proveedores de Servicios: Supabase y Google (solo para almacenamiento y sincronización)
  • Cumplimiento Legal: Si es requerido por ley o para proteger nuestros derechos legales
  • Consentimiento del Usuario: Con su autorización explícita

6. Retención de Datos

Conservamos su información personal mientras:

  • Su cuenta esté activa en el sistema
  • Sea necesario para proporcionar nuestros servicios
  • Sea requerido por obligaciones legales o regulatorias

Puede solicitar la eliminación de sus datos en cualquier momento contactándonos.

7. Sus Derechos

Usted tiene derecho a:

  • Acceder a sus datos personales almacenados en nuestro sistema
  • Rectificar información incorrecta o incompleta
  • Eliminar sus datos personales (derecho al olvido)
  • Oponerse al procesamiento de sus datos
  • Portabilidad de datos en formato estructurado
  • Revocar el acceso a Google Sheets en cualquier momento

8. Seguridad

Implementamos medidas técnicas y organizativas para proteger su información:

  • Conexiones seguras HTTPS/SSL
  • Autenticación OAuth 2.0
  • Encriptación de datos sensibles
  • Acceso restringido a datos mediante políticas de seguridad
  • Monitoreo continuo de seguridad
  • Actualizaciones regulares de seguridad

9. Cookies y Tecnologías Similares

Nuestra aplicación utiliza almacenamiento local del navegador para:

  • Mantener la sesión del usuario
  • Guardar preferencias de configuración
  • Almacenar tokens de autenticación temporales

No utilizamos cookies de terceros para rastreo o publicidad.

10. Menores de Edad

Nuestra aplicación no está dirigida a menores de 18 años. No recopilamos intencionalmente información de menores sin el consentimiento parental.

11. Cambios a esta Política

Nos reservamos el derecho de actualizar esta política de privacidad. Los cambios significativos serán notificados mediante:

  • Actualización de la fecha de "Última actualización"
  • Notificación en la aplicación
  • Correo electrónico (si corresponde)

12. Revocación de Acceso a Google

Puede revocar el acceso de nuestra aplicación a su cuenta de Google en cualquier momento:

  1. Visite https://myaccount.google.com/permissions
  2. Busque "Sistema de Gestión QR"
  3. Haga clic en "Quitar acceso"

13. Cumplimiento Legal

Esta aplicación cumple con:

  • Ley de Protección de Datos Personales (Perú) - Ley N° 29733
  • GDPR (Reglamento General de Protección de Datos - UE)
  • Google API Services User Data Policy
